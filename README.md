# NeoPix-CppRemake ❤️
My pixel art app (Neo pix) remade in C++.

<h2>Controls</h2>
-Add pixel: <span style="color:blue">LClick</span> <br>
-Remove pixel: RClick <br>
-Clear canvas: ESC <br>
-Add color: MiddleClick + look in the console <br>
